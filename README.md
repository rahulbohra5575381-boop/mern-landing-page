# MERN Landing Page 🚀

## 🔗 Live Demo
https://rahulbohra5575381-boop.github.io/mern-landing-page/

## 📌 Project Description
This is a responsive landing page for a MERN stack course platform.  
It showcases course curriculum, features, and pricing in a clean and modern UI.

## 🎯 Features
- Responsive design (mobile + desktop)
- Smooth scrolling navigation
- Course cards with pricing
- Curriculum section (What you'll learn)
- Clean UI with hover effects & transitions

## 🛠 Tech Stack
- HTML
- CSS (Flexbox + Media Queries)

## 🤖 AI / Automation Idea
This project can be extended with:
- Chatbot for student queries (FAQ support)
- Auto-response system for course inquiries
- Smart recommendation system for courses

## 🌍 Use Case (NGO / Healthcare Adaptation)
This same UI can be adapted for:
- Patient support form
- Volunteer registration
- Healthcare awareness platform
