## 🔗 Live Demo

## 📌 Project Description
This is a simple landing page for a MERN stack course platform.  
It is designed to showcase course details, curriculum, and pricing in a clean UI.

## 🎯 Features
- Responsive design (mobile + desktop)
- Smooth scrolling navigation
- Course cards with pricing
- "What You'll Learn" section
- Hover effects and modern UI

## 🛠 Tech Stack
- HTML
- CSS (Flexbox, Media Queries)

## 📚 What I Learned
- Layout design using Flexbox
- Creating responsive websites
- Adding hover effects and transitions
- Structuring a landing page UI
